IV

How It Maps

Single, committed world

The underworld is dark by nature, so this ships as one deliberate dark theme — no light mode to betray the abyss.

Parchment is sacred

Cream + --ink appear only inside narration scrolls. Everything else lives on the abyss ground.

Teal is the Oracle's voice

One accent carries meaning: teal = the Oracle & success, crimson = combat, ember = danger & fire, amethyst = arcane. Gold is trim.

Drops into your tokens

Replaces tokens.css — same variable names, new values — so app.css restyles with almost no structural change.
