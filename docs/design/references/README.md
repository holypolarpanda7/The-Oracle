# Design references — drop art here

Put visual references for The Oracle's interface in **this folder**, then tell me
they're here. I can open image files directly (PNG / JPG / WEBP / screenshots)
and read them visually, so this is the highest-bandwidth way to align on a look.

## What helps most
- **Screenshots of games/apps** whose UI you love (in-game HUD, menus, character
  sheets). Hades II, Disco Elysium, Baldur's Gate 3, Pillars of Eternity, etc.
- **Color/mood boards** — even a single image that captures the palette.
- **Anything with the vibe**: candle-lit, warm, "arcane instrument", parchment,
  gilded, moody — whatever "cooler looking" means to you.

## How to name them
Descriptive filenames help me reason about them, e.g.:
- `hades2-boon-menu.png`
- `palette-candlelit-amber.jpg`
- `bg3-character-sheet.png`
- `NO-too-flat-and-blue.png`  ← negative examples ("not this") are just as useful

## Also useful (no image needed)
- **Exact hex colors** you want (e.g. `#1a1410 bg, #e8b923 accent`).
- **Games/films by name** and what specifically about them (the glow? the type?
  the frames?).
- A one-line "feeling" you want the moment someone opens the Activity.

When you've added files, just say "references are in" and I'll view them and turn
them into a concrete palette + UI-token set for the Activity, then restyle it.
